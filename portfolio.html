UWaterloo Mars Rover Team

Student Design Team
Description

Role: Team Lead (formerly Embedded Software Developer)

The UW Mars Rover named Arthrobot is created in hopes to compete in the University Rover Challenge (held in Hanksville, Utah) and European Rover Challenge (held in Poland). The Arthobot is a four-wheel independent steering exploration robot that can travers through rough terrain, analyze soil samples for signs of life, assist astronauts in various tasks, and manipulate equipment on-site equipment using its fine motion manipulator. Development of the rover is done on the ROS (Robotics Operating System) Open Source framework. A demonstration of the robots capabilities can be seen here as narrated by me.

Contributions
  • Research and developed four-wheel independent steering system and swerve drive algorithm
  • Developed on-board multiaxis camera control system
  • Designed and created Arduino shield-like PCB to simply wiring for servos, sensors and allow for power extention of Orion RobotClaw Motor Controllers
  • Implemented safety features and bumper sensors to prevent harmful overextention of the rover's arm
  • Developed display to show crucial data while performing field tests
  • Implemented smart power distribution system allow safe power usage of the batteries on the rover

UWaterloo Robot Racing Team

Student Design Team
Description

Role: Autonomy and Controls Development

The UWaterloo Robot Racing Team was created to promote research in autonomous mobile robotics and aspires to compete in the International Autonmous Robot Racing Challenge. Our robot can autonmously navigate through a circuit of pylons and compete in a one-on-one drag race using onboard LIDAR and GPS system. The robot is also outfitted with encoders whose data is processed in PID algorithms.

Contributions

  • Developed steering algorithm
  • Created embedded software for onboard LCD module used for testing and displaying sensor data
  • Improved high level control algorithm used to autonoumously drive the robot through a circuit

Mini Desktop Arm

Personal Project
Description

This is a small robotic arm you can use to help you sort out your messy desk! The robot has three different modes: Operator Mode, Teach and Autonomous Mode. Operator mode allows the user to control the robot's joint using a small controller board. Teach mode records the user's movements on the controller and autonomous mode repeats what it is tauch in a repetitive motions. Future developments will allow the robot to use vision algorithm to sort individual items (ex. pens, eraser, coins etc.) and will be using a vacuum pump instead of arm.

Materials
  • Arduino Mini Pro 5V
  • 5 x 10k Potentiometers
  • 5V Power Source
  • Wires, Button, Switch
  • Plastic chassis
  • LCD Monitor for testing

Old Wooden Bluetooth Music Player

In Progress Projects
Description

I took a part an old wooden radio and re-connected its speakers to an external audio amplifier. The amplifier currently has female audio jack that can have any MP3 music player plugged into it. It will eventually be replaced by a bluetooth module to receive audio data and controlled through the Raspberry Pi. The webcam will stream live video to the Raspberry Pi and video processing will be done through OpenCV to allow for gesture playback controls.

Materials
  • Old Radio with working speakers
  • PAM8403 Power Audio Amplifier
  • Raspberry Pi
  • Generic Webcam
  • Bluetooth Module

SPOCK - Smart Padlock

In Progress Projects
Description

I was tired of forgetting combinations and losing keys for padlocks so I decided to prototype a biometric finger scanning smart padlock. The mechancial housing and mechanism will be 3D printed and the finger print scanner will be used to activate a servo motor to trigger the access to the lock. I plan to eventually build a softwware platform ontop of the product such that flexible access can be given to other users who register their fingerprint with the Spock.

Materials
  • Padlock
  • SparkFun Fingerprint Scanner - 5V TTL (GT-511C1R)
  • Small Servo Motor
  • 6V Battery
